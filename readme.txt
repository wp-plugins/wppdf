== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload and unzip `wppdf.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create new pdf publication by going to new-> publications and entering title and selecting pdf file  and thumbnail image
